[Top3 여행지의 관광코스 소개]


베트남(Vietnam)


  • 베트남의 역사와 야경을 동시에, 호치민 3일:

    • "메콩 델타투어" → "호치민 시티투어" → "사이공 나이트투어"


  • 푸른 해변과 놀이공원, 다낭 4일:

    • "다낭대성당" → "미케비치" → "아시아파크" → "한시장" → "오행산"


  • 유럽식 건축 양식을 자연환경과 둘러보는 하노이 4일:

    • "하노이 구시가지" → "호안끼엠 호수" → "하노이 요셉 성당" → "베트남 미술관" → "호치민의 묘"





태국(Thailsand)

  • 시간없는 여행자를 위한, 방콕+파타야 4박 5일:

    • "카오산로드" → "야시장" → "사원(왓 프라께오→왓아룬→왓 포)" → "파타야 산호섬"


  • 유유자적 치앙마이를 중심으로, 태국북부 6일:

    • "트래킹코스" → "빠이 마을" → "전통 사원(왓 프라씽)" → "도이수셉 국립공원" → "쇼핑센터/노천카페" → "코끼리 자연공원"


  • 푸켓 해변을 중심으로 한, 태국 남부 일주일:

    • "푸켓 해변(끄라비)" → "팡아만" → "피피 섬" → "파통의 방글라로드"





괌(Guam)

  • 바다를 느끼고 싶다면 해변으로!:

    • "투몬비치" → "돌핀 크루즈(바다낚시→스노쿨링)" → "수상 엑티비티"


  • 열대우림과 괌 원주민들의 삶을 느껴보고 싶다면 정글로!:

    • "정글리버크루즈" → "차모로족 전통문화체험(카라바오 먹이주기→수공예품 만들기) → "전통 사원(왓 프라씽)" → "불쇼"


  • 즐길거리가 필요하다면 시내로!:

    • "괌 쇼핑센터" → "레스토랑" → "더비치바 디너쇼"
